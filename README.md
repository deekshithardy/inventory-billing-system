# 📦 Inventory & Billing Management System (MERN Stack)

🔗 **Live Project:** https://inventory-frontend-blond.vercel.app

---

## 🚀 Project Overview

A full-stack **Inventory & Billing Management System** built using the MERN stack.  
It enables efficient management of products, customers, and automated invoice generation with secure authentication.
